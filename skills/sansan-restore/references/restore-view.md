# 商业诊断恢复展示格式

只从存档提取已有字段，保持一屏可读。

```markdown
## 上次诊断到这里

**项目**：
**存档时间**：
**标题**：
**当时状态**：进行中 / 已结论 / 已放弃
**当时来自**：

### 当时确认的事实

- ...

### 当时的最大瓶颈

- ...

### 已排除的方向

- ...

### 待验证假设

- ...

### 上次的7天动作

- ...

### 应该回流的数据

- ...

### 上次推荐的下一步

...

> 以上是当时的诊断状态，不自动代表当前情况。
```

随后复用当前对话已有的新事实；没有新事实时，只问一次“上次之后，实际发生了什么变化？”
