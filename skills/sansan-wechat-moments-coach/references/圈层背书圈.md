# 圈层背书圈

## 什么时候使用

素材核心是受邀、获奖、合作、入选、登台或进入特定圈层。此类内容用真实事件建立身份与信任，不把圈层包装成能力和结果的充分证明。

## 必要输入

- 活动、品牌或组织准确名称；
- 公开报名还是定向邀请；
- 使用者获得什么真实身份；
- 对方如何正式称呼；
- 使用者与对方的真实关系；
- 为什么这件事有意义；
- 是否允许公开邀请函和现场内容。

## 核心模板

```text
品牌或个人的重要节点
→ 使用者收到的真实邀请或认可
→ 与普通活动相比为什么不同
→ 主办方正式定义的身份
→ 使用者的个人感受
→ 克制表达关系和分量
```

## 标题方向

- 【品牌】走到【节点】，我的名字出现在邀请函上。
- 收过不少邀请，但这次的【真实关系意义】不一样。
- 被邀请参加【活动】，我最看重的不是头衔。

## 边界

- 普通公开报名不能包装成私人邀请；
- “自己人、贵宾、核心圈层”必须符合主办方真实定义；
- 稀缺性需要真实人数或筛选机制；
- 不贬低未受邀者；
- 不把合影自动写成合作或背书；
- 邀请函需遮盖二维码、地址、编号和其他参与者信息；
- 品牌名称、设计和现场内容可能需要公开许可。

默认不加销售钩子。重点是事件、关系和个人感受。
