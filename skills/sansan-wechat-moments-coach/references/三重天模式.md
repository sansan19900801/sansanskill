# 三重天模式

## 目标

从同一份真实素材生成四条不同层次的朋友圈，让使用者在不重复、不编造的前提下，同时获得转化、价值观和长期信任内容。

## 必要输入

第一轮只让使用者提供：

- 素材属于观点、方法、产品发售、圈层背书，还是业务事件；
- 真实发生了什么；
- 最想表达的判断是什么。

可以直接照着说：

> 这份素材属于______；真实发生的事情是______；我最想表达的判断是______。

对应产品、原话、动作、结果和证据优先从当前素材中提取。信息不足时只补问最关键的一项，不为凑四条虚构材料。

三重天朋友圈不强制要求截图；纯文字素材足够真实、清楚时可以直接生成。

## 固定配比

### 第一重天 × 2：人聚财

面向行动和转化，但不强推。

第一条优先写：

```text
具体问题
→ 真实事件
→ 解决动作
→ 可验证结果
→ 轻量咨询入口
```

第二条优先写：

```text
客户原话或结果
→ 使用前后变化
→ 方法价值
→ 同类客户自检
```

### 第二重天 × 1：地聚人

表达价值观，筛选同频者。

```text
真实事件
→ 使用者为什么这样做
→ 与另一种选择的差别
→ 自己坚持的原则
→ 不强迫他人认同
```

### 第三重天 × 1：天聚福

从事件提炼长期规律或底层理解。

```text
一个具体细节
→ 发现的反复模式
→ 成立条件
→ 不适用边界
→ 一句克制结论
```

## 输出格式

```markdown
## ① 第一重天｜角度名称

【正文】

评论区：【必要时写】

配图：【建议】
```

依次输出①②③④。四条不附带长篇理论分析。

## 红线

- 不为四条内容添加新客户、新数字、新对话和新情绪；
- 不把第一重天写成假稀缺和强催付；
- 不把第二重天写成道德评判；
- 不把第三重天写成脱离事件的宏大鸡汤；
- 四条必须有不同角度，不做同义改写；
- 素材只够两条时，明确说明并只生成两条高质量内容。
