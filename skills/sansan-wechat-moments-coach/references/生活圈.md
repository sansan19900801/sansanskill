# 生活圈

## 什么时候使用

素材核心是家庭、旅行、天气、日常劳动、自然场景或个人感受。生活圈让读者看见真实的人，不承担每条都成交的任务。

## 必要输入

第一轮只让使用者提供：

- 什么时间；
- 什么地点；
- 和谁；
- 发生了什么事。
- 本次生活事件的照片或相关截图。

可以直接照着说：

> ______时间，我在______，和______一起发生了______。这是当时的照片或相关截图。

没有照片或相关截图时先要求补图，不生成生活圈。具体细节和真实感受优先从使用者后续叙述或照片中提取；没有事实支撑时不自行编造。

## 模板选择

### A. 预期与结果反转

```text
大家原本期待或担心什么
→ 自己做了哪些准备
→ 实际结果完全不同
→ 一个现场细节
→ 真实复杂情绪
→ 轻感悟
```

### B. 同一场景的时间反差

```text
平时什么时候来到这里
→ 这次改变时间或角度
→ 进入场景的动作
→ 两三个感官细节
→ 与平时的氛围对比
→ 一个轻量决定
```

### C. 家庭陪伴

```text
陪谁做了什么
→ 自己完成的照顾动作
→ 家人的真实反应
→ 哪个瞬间触发感受
→ 只表达个人理解
```

### D. 突发小插曲

```text
日常活动
→ 突发变化
→ 家人的连续动作
→ 意外反转
→ 相互互动
→ 简短幸福感
```

## 写作要求

- 多写可见、可听、可触的细节；
- 允许轻度比喻和拟人，但不编造场景；
- 不强行植入产品、二维码和商业道理；
- 不把个人家庭选择变成道德标准；
- 不通过衰老、疾病、失去制造焦虑；
- 涉及孩子、长辈和实时位置时保护隐私；
- 使用网络图片时不得冒充本人生活。

默认不写评论区销售钩子。需要互动时，用自然问题收尾。
