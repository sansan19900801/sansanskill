# 咨询圈

## 什么时候使用

素材核心是潜在客户因为某条内容、案例、介绍或体验主动来问。咨询圈展示“什么人因为什么问题来咨询”，不把咨询自动写成成交。

## 必要输入

第一轮只让使用者提供：

- 他是谁；
- 他为什么会来咨询使用者的产品。
- 客户咨询聊天截图。

可以直接照着说：

> 他是一位______，因为______来咨询我的______。这是客户咨询截图。

没有咨询截图时先要求补图，不生成咨询圈。如果当前对话和截图仍不足以写清事实，再补问一个最关键的问题；不要第一轮同时索取来源、问话、回应和后续进展。

## 核心模板

```text
咨询者真实身份或特征
→ 从哪条内容认识使用者
→ 主动咨询的动作
→ 客户亲口说出的具体问题
→ 一次真实回应、演示或诊断
→ 客户原话或后续行动
→ 该咨询揭示的业务需求
→ 给同类客户一个低压力入口
```

## 标题方向

- 【客户类型】为什么看完这条内容后主动来问？
- 她不是不会【工具】，真正卡住的是【业务问题】。
- 这次咨询让我再次确认：客户真正需要的是【结果】。

## 边界

- 一位大客户或百万博主不能写成“全行业都来问”；
- 身份、粉丝量和公司必须真实并获得允许；
- 客户动作、情绪和引号内容不能编；
- “10秒演示、立刻惊呆”需要真实记录；
- 咨询不等于付款，体验不等于结果；
- 评论区可以邀请描述问题，不强行报名。
